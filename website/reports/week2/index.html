2개 문제 — 양 대신 깊이

김호윤 학생은 다른 방식으로 기여한다. 한 문제 안에 측정 의도를 깊이 박아넣는다.

mat_hoyoon_001
전위(Dislocation) 개념 설명

L2 · materials · rubric 4축 (정확도 · 설명력 · 용어사용 · 한국어품질)

멘토 가이드 예시를 충실히 따른 안전한 첫 시도.

ochem_001 ★
친전자성 첨가 반응 — 프로펜 + HBr

L2 · organic_chemistry · rubric 5축

마르코브니코프 법칙 없이 HOMO/LUMO + VBT 원리만으로 학생을 주생성물까지 유도하는 소크라테스식 튜터링 능력 측정.

ochem_001이 특별한 이유

  1. 3단계 논리 체인 강제. HOMO 위치 특정 → 중간체 생성 → VBT 안정성 비교. 각 단계가 이전 단계의 답을 전제.
  2. must_not_include 5개 정밀. "친전자성 첨가반응", "마르코브니코프", "1차 카보카티온보다 2차 카보카티온이 안정" — 정답을 직접 흘리는 모든 패턴을 정확히 차단.
  3. purpose 필드를 객체로 확장. 핵심_목적 / 이_반응을_선택한_이유 / 측정_역량 — 왜 이 문제를 만들었는지의 설계 의도까지 명시.
  4. 법칙 암기 vs 원리 이해라는 새 측정 축. 기존 평가에 없던 "AI가 학생에게 답이 아닌 추론을 줄 수 있는가"를 한 문제로 깊이 있게 검증.

두 학생이 합쳐져 만드는 것

김태운은 시스템과 양을, 김호윤은 설계와 정밀도를 가져온다. 이 둘이 합쳐지지 않으면 벤치마크는 "단순 비교 도구"에 머물거나 "예쁜 한 문제"에 머문다. 둘 다 있어야 살아있는 벤치마크가 된다.
