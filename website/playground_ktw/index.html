EXPERIMENT №001 · Multi-Model Comparison v0.2 · KTW

같은 문제를
여러 AI에게 동시에.

SAM API로 한 번에 여러 모델을 호출하고 결과를 나란히 비교한다. 본인의 학업 문제를 직접 입력하고, 어느 모델이 더 도움이 되는지 체감한다.

STEP / 01

SAM API 키 입력

API 키는 브라우저의 localStorage에만 저장되며 외부 서버로 전송되지 않는다. 키 발급은 sam.soonsoon.ai에서 받을 수 있다.

STEP / 02

문제 입력

좌측 트리에서 벤치마크 문제를 선택하거나, 직접 입력할 수 있다. 한국어로 자연스럽게 작성하라.

STEP / 03

모델 선택

비교할 모델을 2~5개 선택한다. 각 모델의 응답 속도와 비용은 다르다.

STEP / 04

실행 & 비교

STEP / 05

관찰 노트

어느 모델이 더 도움이 되었는지, 비싸야 잘 푸는지, 한국어 품질은 어떤지 본인의 평가를 작성하라. 이 데이터가 우리 벤치마크의 출발점이다.

  • 한국어 학술 문체로 답하는가? (어색한 번역체는 없는가)
  • 풀이 과정을 단계별로 보여주는가?
  • 참고할 수 있는 출처/공식을 제시하는가?
  • 모르는 것을 솔직하게 인정하는가?
  • 비싼 모델이 정말 더 도움되는가? (가성비)
