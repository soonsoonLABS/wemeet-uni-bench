📜 History Viewer

왼쪽 사이드바의 데이터베이스(DB) 실행 이력을 선택하거나, 로컬 JSON 파일을 업로드하여 벤치마크 모델 응답을 나란히 상세 비교합니다.

💡 사용 가이드

  • DB 결과 불러오기: 로컬 API 서버에 저장된 벤치마크 결과 리스트를 실시간 동기화합니다.
  • JSON 파일 드롭: 이전에 로컬 저장소로 다운로드해두었던 run_*.json 파일을 끌어다 놓으면 즉시 브라우저에서 읽어 대조해 볼 수 있습니다.
  • 반응형 레이아웃: 사이드바 상단의 ◀ 버튼을 누르면 이력 목록이 축소되어 결과를 화면에 더 넓게 표시합니다.
