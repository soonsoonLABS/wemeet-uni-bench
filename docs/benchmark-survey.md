## 기존 벤치마크의 한계 (대학생 관점)

### 1. 언어 편향
- 대부분 영어 기반 → 한국어 학술 맥락 반영 부족
- KMMLU 존재하나 학부 과제 시나리오와는 거리 있음

### 2. 시나리오 괴리
- 단답형/객관식 위주 → 실제 과제는 서술형, 보고서, 코드 프로젝트
- 독립 문제 → 실제는 맥락 연속적 (이전 대화 기반 후속 질문)

### 3. 평가 차원 부재
- "과제 도움" 관점의 평가 없음 (설명력, 단계별 풀이, 참고문헌 제시 등)
- 학습 보조 능력 (개념 설명, 오류 지적, 대안 제시) 미측정

### 4. 난이도 불일치
- GSM8K: 너무 쉬움 (초등 수준)
- GPQA: 너무 어려움 (대학원 전문가 수준)
- 학부 2~4학년 수준의 적절한 난이도 벤치마크 부재
