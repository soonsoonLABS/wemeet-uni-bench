# 프로젝트 비전 및 취지

---

## 왜 이 프로젝트를 하는가

### 문제 인식

AI 모델의 성능을 측정하는 벤치마크는 수십 개가 존재하지만, 정작 **한국 대학생이 학업에서 AI를 쓸 때 "어떤 모델이 나한테 도움이 되는가"를 알려주는 지표는 없다.**

- MMLU 90점이면 내 선형대수 과제를 잘 도와주나? → 모름
- HumanEval 95%면 내 자료구조 과제를 잘 풀어주나? → 모름
- GPQA 92%면 내 재료역학 문제를 설명해주나? → 모름

기존 벤치마크는 AI 연구자들이 모델 간 우열을 가리기 위해 만든 것이지, **학생이 실제로 도움받을 수 있는지를 알려주기 위해 만든 것이 아니다.**

---

## 우리가 만들고 싶은 것

### 학생이 직접 참여하는 살아있는 벤치마크

단순히 연구팀이 문제를 만들고 끝나는 것이 아니라, **실제 학생들이 자신의 과제, 시험 문제, 학습 중 막힌 질문을 계속 제공하면서 성장하는 벤치마크**를 지향한다.

```
학생이 문제 제공 → 벤치마크에 반영 → AI 모델 평가 → 결과 공개 → 학생이 더 나은 모델 선택
     ↑                                                              |
     └──────────────────── 피드백 루프 ─────────────────────────────┘
```

---

## 핵심 취지

### 1. 🎯 학생 중심의 실질적 평가

> "이 AI가 내 과제에 도움이 되는가?"

기존 벤치마크가 묻는 질문: "이 모델이 얼마나 똑똑한가?"
우리가 묻는 질문: **"이 모델이 나한테 얼마나 도움이 되는가?"**

- 정답을 맞히는 것뿐 아니라, 설명을 잘 하는가
- 풀이 과정이 이해하기 쉬운가
- 한국어로 자연스럽게 답하는가
- 모르는 것을 모른다고 말하는가

### 2. 🤝 학생이 만들고 학생이 쓰는 공공재

> "내가 어려웠던 문제가 다른 학생에게도 도움이 된다"

- 학생들이 자신이 풀다가 막힌 문제, 실제 시험 문제, 과제에서 AI에게 물어본 질문을 제공
- 이 문제들이 벤치마크에 반영되어 AI 모델의 "학생 도움 능력"을 측정
- 결과가 공개되어 모든 학생이 자기 상황에 맞는 모델을 선택할 수 있음
- **기여한 학생은 크레딧을 받고, 벤치마크는 계속 풍부해짐**

### 3. 📈 계속 성장하는 살아있는 데이터셋

> "한 번 만들고 끝나는 벤치마크가 아니라, 매 학기 업데이트되는 벤치마크"

기존 벤치마크의 고질적 문제:
- 한 번 공개되면 학습 데이터에 오염됨
- 시간이 지나면 포화되어 변별력 상실

우리의 해결책:
- 매 학기 새로운 문제가 학생들로부터 유입
- 자연스럽게 오염 방지 (새 문제가 계속 추가되니까)
- 시대/교육과정 변화에 맞춰 진화

### 4. 🇰🇷 한국 교육 맥락에 맞는 평가

> "한국 대학에서 실제로 쓰이는 방식으로 평가한다"

- 한국어 학술 문체로 답변하는 능력
- 한국 대학 교재/커리큘럼 기반 문제
- 한국 학생이 실제로 AI에게 묻는 방식의 프롬프트
- 한국 교수가 기대하는 답안 형식에 맞는 출력

### 5. 🌐 오픈소스로 누구나 참여 가능

> "서울대에서 시작하지만, 모든 대학의 학생이 기여할 수 있다"

- 문제 제출 → 검수 → 반영 파이프라인을 오픈
- 다른 대학, 다른 전공의 학생도 자기 분야 문제를 추가
- 대학별/전공별 세분화된 결과 제공 가능
- 궁극적으로 "한국 대학생 AI 활용 생태계"의 공공 인프라

### 6. 💡 AI 서비스 개선에 직접 기여

> "벤치마크 결과가 실제 AI 서비스를 더 좋게 만든다"

- SAM 플랫폼에서 "학생 도움 점수"로 모델 추천에 활용
- 모델 제공사가 한국 학생 시나리오에서의 약점을 파악하고 개선
- 학생 → 문제 제공 → 평가 → 모델 개선 → 학생에게 더 나은 서비스

## 기대 효과

### 단기 (이번 프로젝트)
- 6개 카테고리 × 4단계 난이도의 체계적 벤치마크 초안 완성
- SAM 플랫폼 모델 대상 첫 벤치마크 실행 및 결과 공개
- 프로젝트 구조와 기여 방법 문서화

### 중기 (1~2학기 후)
- 다른 대학/전공 학생의 문제 기여 시작
- 벤치마크 문제 수 200+ 확보
- SAM 플랫폼에 "Student Score" 공식 반영

### 장기 (1~2년)
- 한국 대학생 AI 벤치마크의 사실상 표준(de facto standard)
- 매 학기 자동 업데이트되는 라이브 벤치마크
- AI 모델 제공사가 한국 학생 시나리오를 고려하여 모델 개선
- 논문 발표 및 학술적 기여

## 참여 방법 (학생 기여자)

### 문제 제공하기
1. 자신이 풀다가 막힌 문제, AI에게 물어봤던 질문을 정리
2. GitHub Issue로 제출 (템플릿 제공)
3. 검수 후 벤치마크에 반영
4. 기여자로 크레딧 등록

### 기여 가능한 것들
- 실제 과제/시험에서 가져온 문제 (저작권 주의)
- AI에게 물어봤는데 답이 별로였던 질문
- "이런 건 잘 해주면 좋겠다" 시나리오
- 채점 기준에 대한 피드백
- 기존 문제의 난이도/품질 평가

---

## 한 줄 요약

> **"학생이 만들고, 학생이 쓰고, 학생을 위한 AI 벤치마크"**

기존 벤치마크가 "AI가 얼마나 똑똑한가"를 측정한다면,
우리는 **"AI가 학생에게 얼마나 도움이 되는가"**를 측정한다.
