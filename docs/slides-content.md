# 슬라이드 콘텐츠 정리

> 5~7장 분량, 핵심 메시지 중심으로 구성

---

## 슬라이드 1: 타이틀

**제목**: AI가 똑똑한 건 알겠는데, 내 과제는 잘 도와주나?

**부제**: 한국 대학생을 위한 AI 모델 벤치마크 프로젝트

**하단 정보**:
- 서울대학교 WE-Meet × 순순팩토리
- wemeet-uni-bench

---

## 슬라이드 2: 문제 제기

**제목**: 벤치마크 점수가 높으면 나한테 도움이 되나?

**핵심 메시지**:

| AI 회사가 자랑하는 것 | 학생이 실제로 궁금한 것 |
|---------------------|---------------------|
| MMLU 90% | 내 선형대수 과제 풀어줄 수 있어? |
| HumanEval 95% | 내 자료구조 코드 디버깅 해줘 |
| GPQA 92% | 재료역학 개념 쉽게 설명해줘 |
| Arena ELO 1위 | 한국어로 레포트 써줄 수 있어? |

**한 줄**: 기존 벤치마크는 AI 연구자를 위한 것이지, 학생을 위한 것이 아니다.

## 슬라이드 4: 기존 벤치마크의 한계

**제목**: 기존 벤치마크가 놓치는 것들

**4가지 핵심 한계** (아이콘 + 짧은 설명):

1. **객관식만 있다**
   - 실제 과제: 서술형, 코딩 프로젝트, 레포트
   - 벤치마크: 4지선다 정답 고르기

2. **답만 채점한다**
   - 학생이 원하는 것: 단계별 풀이, 이해하기 쉬운 설명
   - 벤치마크: 최종 답 맞으면 끝

3. **영어뿐이다**
   - 한국어 학술 문체? 한국 교재 맥락? → 측정 안 함

4. **"도움이 되는가"를 안 묻는다**
   - 맞는 답 ≠ 도움이 되는 답
   - 설명력, 참고자료 제시, 오류 인정 → 아무도 측정 안 함

---

## 슬라이드 5: 우리가 만드는 것

**제목**: 학생이 만들고, 학생이 쓰는 AI 벤치마크

**핵심 구조 (순환 다이어그램)**:

```
    ┌─────────────────────────────────────┐
    │                                     ▼
학생이 문제 제공  →  벤치마크 반영  →  AI 모델 평가  →  결과 공개  →  더 나은 선택
(내 과제, 내 질문)    (검수 후 등록)    (SAM 모델 실행)   (점수 공개)    (학생에게 환원)
    ▲                                                                    │
    └────────────────────────── 피드백 루프 ─────────────────────────────┘
```

**차별점 3가지**:
- 📝 풀이 과정과 설명력을 평가한다
- 🇰🇷 한국어 + 한국 대학 맥락이다
- 🔄 학생이 계속 문제를 제공하며 성장한다

---

## 슬라이드 6: 평가 체계

**제목**: 어떻게 평가하나

**4개 축** (파이차트 또는 바 형태):

| 평가 축 | 비중 | 측정하는 것 |
|---------|------|------------|
| 정확성 | 40% | 답이 맞는가 |
| 유용성 | 30% | 바로 과제에 쓸 수 있는가 |
| 설명력 | 20% | 이해하기 쉽게 설명하는가 |
| 한국어 품질 | 10% | 자연스러운 학술 한국어인가 |

**6개 카테고리**:
학술 글쓰기 · 이공계 문제 풀이 · 코딩 과제 · 리서치 보조 · 데이터 분석 · 학습 보조

**4단계 난이도**: L1(1학년) → L2(2학년) → L3(3학년) → L4(4학년/졸업)

---

## 슬라이드 7: 비전 — 어디까지 갈 수 있나

**제목**: 한국 대학생 AI 활용의 공공 인프라

**타임라인 (3단계)**:

**지금** (이번 프로젝트)
- 서울대 컴공 + 재료공학 기반 50+ 문제
- SAM 모델 대상 첫 벤치마크 실행
- 오픈소스 공개

**다음** (1~2학기 후)
- 다른 대학/전공 학생 참여 시작
- 200+ 문제, SAM "Student Score" 공식 반영
- 모델 선택 가이드 제공

**궁극적으로**
- 매 학기 자동 업데이트되는 라이브 벤치마크
- 한국 대학생 AI 벤치마크의 사실상 표준
- AI 서비스가 한국 학생 시나리오를 고려하여 개선

**마무리 한 줄**:

> "AI가 얼마나 똑똑한가"가 아니라,
> **"AI가 나한테 얼마나 도움이 되는가"**를 측정한다.

## 디자인 노트 (슬라이드 제작 시 참고)

- **톤**: 학생 친화적, 직관적, 무겁지 않게
- **컬러**: 순순팩토리 브랜드 + 서울대 블루 계열 조합
- **폰트**: 한글 가독성 좋은 것 (Pretendard, SUIT 등)
- **비주얼 핵심**:
  - 슬라이드 3의 난이도 스펙트럼 → 한눈에 갭이 보여야 함
  - 슬라이드 5의 순환 구조 → 학생 참여가 핵심임을 시각화
  - 슬라이드 6의 4축 평가 → 기존(답만 채점)과의 차이 강조
- **분량**: 발표용이면 7장 그대로, SNS/소개용이면 2-5번 슬라이드 4장으로 압축 가능
